Detailed Description

Render a single volumetric line

Based on the Volumetric lines algorithm by Sebastien Hillaire http://sebastien.hillaire.free.fr/index.php?option=com_content&view=article&id=57&Itemid=74

Thread in the Unity3D Forum: http://forum.unity3d.com/threads/181618-Volumetric-lines

Unity3D port by Johannes Unterguggenberger johan.nosp@m.nes..nosp@m.unter.nosp@m.gugg.nosp@m.enber.nosp@m.ger@.nosp@m.gmail.nosp@m..com

Thanks to Michael Probst for support during development.

Thanks for bugfixes and improvements to Unity Forum User "Mistale" http://forum.unity3d.com/members/102350-Mistale

Shader code optimization and cleanup by Lex Darlog (aka DRL) http://forum.unity3d.com/members/lex-drl.67487/
